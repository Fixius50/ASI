Fase 1: Sanitización

Preparación y limpieza del entorno

Esta acción detendrá todos los servicios de IA y eliminará configuraciones previas de OpenCode y Ollama.

Terminal - bash
# 1. Detener servicios
pkill OpenCode || true
pkill ollama || true

# 2. Eliminar residuos y apps
sudo rm -rf /Applications/OpenCode.app
rm -rf ~/.opencode
rm -rf ~/.ollama

echo "☑ Entorno limpio y listo."

Fase 2: Instalación Automatizada

Despliegue mediante Script Bash

Paso 1: Crear Script
Script Oficial
setup_full.sh
#!/bin/bash
set -e

# --- VARIABLES OFICIALES ---
MODEL_ID="gpt-oss:20b"
# URL Release GitHub
OPENCODE_URL="https://github.com/anomalyco/opencode/releases/latest/download/OpenCode-Darwin-arm64.dmg"

# 1. INSTALACIÓN DE OLLAMA
curl -fsSL https://ollama.com/install.sh | sh

# 2. CONFIGURACIÓN RED
launchctl setenv OLLAMA_HOST "0.0.0.0"

# 3. OPENCODE & MODELO
ollama pull $MODEL_ID
